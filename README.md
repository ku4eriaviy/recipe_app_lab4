# Информация о рецептах XML

Django-приложение для хранения рецептов в формате XML.

## Функционал

- Ввод рецепта через HTML-форму → сохранение в XML.
- Загрузка XML-файлов на сервер.
- Проверка валидности XML (структура).
- Безопасное имя файла (генерация UUID).
- Отображение всех загруженных рецептов на сайте.
- Уведомления об успехе/ошибке.
